I developed a fully functional digital adaptation of Durak, the popular Russian card game, using Unity and C#. This project focused on translating complex, turn-based card logic into a seamless software experience, featuring a robust game state manager to handle attacks, defenses, and the "fool" condition. To ensure an engaging user experience, I designed an interactive UI with responsive input handling and implemented a single-screen multiplayer mode. By utilizing a pass and play mechanic, I optimized the interface for local competition, delivering a polished and intuitive flow that maintains the tension and strategy of the original tabletop game.
